## H2.live 

### Information

This application uses hydrogen station data from H2.LIVE, European information portal for hydrogen mobility.

### Data Overview

- Live station information pulled every 2 minutes from public hydrogen stations across Europe
- Covers stations providing 350 bar (mainly for buses and commercial vehicles) and/or 700 bar (mainly for passenger cars)

### Key Data Points

Stations are displayed with:
- Location on map
- Availability status
- Construction status

### Data Source

All data is sourced from H2.LIVE. For comprehensive information about hydrogen stations, including:
- Station addresses
- Operators
- Opening hours
- Contact details
