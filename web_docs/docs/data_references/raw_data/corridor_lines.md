## Corridor lines - Ten-T 
 
- Layer title: Line data from Ten-T affecting the Hytruck corridors. 
- Layer name: corridor_lines.gpkg 
- Sub-layers: None 
- Description: Vector line dataset containing Ten-T corridor data for the HyTruck project from the Trans-European Transport Network.  
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 4.0151849996335640,35.8147479898280494 : 28.3378675002941591,68.5163204984550589 
- Temporal extent: 14/09/2023 (time-instant) 
- CRS: ETRS89-extended / LAEA Europe (epsg:3035) 
- Encoding: UTF-8 
- Scale/Denominator: Unknown. 
- Topic category: Transportation 
- Keywords: infrastructure, roads, railways, waterways 
- List of input datasets: inland waterways, roads and railways from Ten-T. 
- Attributes: 
	[corridor] 
	Data type: String (50) 
	Description: Name of the corridor. 
	[type] 
	Data type: String (15) 
	Description: Describes the type of line (inland waterway, road, or railway). 
- Date: Accessed on 14/09/2023 through and older ArcGIS REST Service. Currently available via 
https://webgate.ec.europa.eu/getis/rest/services/TENTec/tentec_public_services_v4/MapServer/ 
- Quality/Lineage: The line datasets of inland waterways, roads, and railways have been merged for the following corridors from TenT: 
1) Scandinavian-Mediterranean, 2) North Sea - Baltic, and 3) Baltic - Adriatic.
