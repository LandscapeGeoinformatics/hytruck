## Wind farms - Global Energy Monitor 
 
- Layer title: Wind power farms from the Global Energy Monitor 
- Layer name: wind.gpkg 
- Sub-layers: None 
- Description: Vector point dataset containing wind farms from the Global Energy Monitor 
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 5.9880000000000004,47.6756999962405743 : 30.2126999999999875,67.6371999989525534 
- Temporal extent: 12/2023 (time-instant) 
- CRS: WGS84 (epsg:4326) 
- Encoding: UTF-8 / latin1 
- Scale/Denominator: Unknown 
- Topic category: Structure 
- Keywords: wind, infrastructure, energy 
- List of input datasets: None 
- Attributes:  
	[name]  
	Data type: String (50) 
	Description: Text describing the name of the solar farm. 
	[owner] 
	Data type: String (50) 
	Description: Text describing the name of the company owning the solar farm. 
	[capacity_mw] 
	Data type: Integer 
	Description: Refers to the power of the solar farm in MW. 
	[accuracy] 
	Data type: String (11) 
	Description: Refers to the accuracy of the point data, either approximate or exact. 
	[gem_id] 
	Data type: String (7) 
	Description: Refers to the original source unique ID of the solar farm facility.  
- Date: Retrieved Global Energy Monitor on 11/02/2024. 
- Quality/Lineage: The excel large scale solar data has been imported as points using Lat/Long fields and clipped by HyTruck extent, 
and the attributes have been cleaned and optimised.
