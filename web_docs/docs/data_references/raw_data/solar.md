## Solar farms 
 
- Layer title: Solar power farms 
- Layer name: solar.gpkg 
- Sub-layers: None 
- Description: Vector point dataset containing solar farms from the Global Energy Monitor 
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 6.2308998107910201,48.0874938964844034 : 28.1599006652831996,65.5528030395507955 
- Temporal extent: 12/2023 (time-instant) 
- CRS: WGS84 (epsg:4326) 
- Encoding: UTF-8 / latin1 
- Scale/Denominator: Unknown 
- Topic category: Structure 
- Keywords: solar, infrastructure, energy 
- List of input datasets: None 
- Attributes:  
	[name]  
	Data type: String (50) 
	Description: Text describing the name of the wind farm. 
	[owner] 
	Data type: String (50) 
	Description: Text describing the name of the company owning the wind farm. 
	[capacity_mw] 
	Data type: Integer 
	Description: Refers to the power of the wind farm in MW. 
	[accuracy] 
	Data type: String (11) 
	Description: Refers to the accuracy of the point data, either approximate or exact. 
	[gem_id] 
	Data type: String (7) 
	Description: Refers to the original source unique ID of the wind farm facility.  
- Date: Retrieved Global Energy Monitor on 11/02/2024. 
- Quality/Lineage: The excel large scale wind data has been imported as points using Lat/Long fields and clipped by HyTruck extent, 
and the attributes have been cleaned and optimised.
