## Seashore - Eurostat 
 
- Layer title: Seashore line of the HyTruck partner countries. 
- Layer name: seashore.gpkg 
- Sub-layers: None 
- Description: Vector line dataset from Eurostat (GISCO) representing the shoreline affecting HyTruck partner countries (Finland, Estonia,  
Latvia, Lithuania, Poland, Germany, Denmark, and Sweden). 
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 5.8676973324401356,47.2701139962005215 : 31.5860086947638870,70.0921365900435376 
- Temporal extent: 03/04/2020 (time-instant) 
- CRS: ETRS89-extended / LAEA Europe (epsg:3035) 
- Encoding: UTF-8 
- Scale/Denominator: 1/1,000,000 
- Topic category: Oceans 
- Keywords: Seashore, coast, borders. 
- List of input datasets: Eurostat (GISCO) country borders. 
- Attributes: None 
- Date: Published 03/04/2020. 
- Quality/Lineage: The Eurostat (GISCO) borders for the HyTruck partner countries has been converted into lines and difference tool has 
been used with the bordering countries in order to get the shoreline of the area of interest.
