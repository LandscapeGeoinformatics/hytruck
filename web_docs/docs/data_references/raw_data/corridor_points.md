## Corridor points - Ten-T 
 
- Layer title: Point data from Ten-T affecting the Hytruck corridors. 
- Layer name: corridor_lines.gpkg 
- Sub-layers: None 
- Description: Vector point dataset containing Ten-T corridor data for the HyTruck project from the Trans-European Transport Network.  
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 4.2939910000817676,35.8193304903434608 : 27.1796385003121230,68.4302869982461033 
- Temporal extent: 14/09/2023 (time-instant) 
- CRS: ETRS89-extended / LAEA Europe (epsg:3035) 
- Encoding: UTF-8 
- Scale/Denominator: Unknown. 
- Topic category: Transportation 
- Keywords: infrastructure, airport, terminal, railway station, port 
- List of input datasets: airports, ports and rail-road terminals from Ten-T. 
- Attributes: 
	[corridor] 
	Data type: String (50) 
	Description: Name of the corridor. 
	[type] 
	Data type: String (20) 
	Description: Describes the type of point (airport, rail-road terminal, or port). 
- Date: Accessed on 14/09/2023 through and older ArcGIS REST Service. Currently available via 
https://webgate.ec.europa.eu/getis/rest/services/TENTec/tentec_public_services_v4/MapServer/ 
- Quality/Lineage: The point datasets of ariports, rail-road terminals, and ports have been merged for the following corridors from TenT: 
1) Scandinavian-Mediterranean, 2) North Sea - Baltic, and 3) Baltic - Adriatic.
