## Gas pipelines - GEM 
 
- Layer title: Gas pipelines from the Global Energy Monitor.  
- Layer name: gas_pipelines.gpkg 
- Sub-layers: None 
- Description: Vector line dataset containing gas pipelines under the project Global Gas Infrastructure Tracker from the  
Global Energy Monitor.  
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 5.8996706770435408,47.5295008155366432 : 28.7644999999999946,61.6599999986204637 
- Temporal extent: 10/02/2024 (time-instant) 
- CRS: ETRS89-extended / LAEA Europe (epsg:3035) 
- Encoding: UTF-8 
- Scale/Denominator: Unknown 
- Topic category: Structure 
- Keywords: gas, infrastructure, energy 
- List of input datasets: None 
- Attributes:  
	[name]  
	Data type: String (75) 
	Description: Text describing the name of the gas pipeline. 
	[owner] 
	Data type: String (100) 
	Description: Text describing the name of the company/ies owning the solar farm, including percentages in some cases. 
	[capacity] 
	Data type: Real 
	Description: Refers to the capacity of the gas pipeline in billions of cubic meters per year. 
	[status] 
	Data type: String (10) 
	Description: Refers to the status of the pipeline (Operating, construction, preconstruction,retired, shelved,canceled,  
	annouced, and mothballed) 
- Date: Retrieved from Global Energy Monitor on 10/02/2024. 
- Quality/Lineage: The data has been clipped by HyTruck extent, and attributes cleaned and optimised.
