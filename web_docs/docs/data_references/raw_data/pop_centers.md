## Population Centers - JRC 
 
- Layer title: Population centers dataset from GHSL - Global Human Settlement Layer (JRC-EU) 
- Layer name: pop_centers.gpkg 
- Sub-layers: germany, denmark, poland, lithuania, latvia, estonia, finland, sweden. 
- Description: Vector polygon dataset containing population centers from the European Commission and Joint Research Center (JRC) 
based on GHS-UCDB R2019A and GHSL 1km grids. 
- Format: GeoPackage (gpkg) 
- Version: 1.4.0 
- Geographical extent (xmin,ymin : xmax:ymax): 5.9958402910000643,47.5506222487093950 : 27.7349598787954932,65.0586221301483931 
- Temporal extent: 01/01/1975-30/12/2018 (time-period) 
- CRS: WGS84 (epsg:4326) 
- Encoding: UTF-8 
- Scale/Denominator: None 
- Resolution: 1km vector grids 
- Topic category: Utilities communication 
- Keywords: infrastructure, electricity, plant 
- List of input datasets: None 
- Attributes:  
	[area]  
	Data type: Real 
	Description: Polygon area of the population center in square kilometers (km2). 
	[name] 
	Data type: String (50) 
	Description: Geographical name of the population center / city.  
- Date: 2019 (published) & 2022-06-24 (modified). 
- Quality/Lineage: The data has been compared clipped with the partner countries of HyTruck and attributes have been cleaned.
